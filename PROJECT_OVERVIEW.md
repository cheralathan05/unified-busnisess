## 🚀 Deployment Guide

### Deploy to Vercel (Recommended)
```bash
# Push to GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/[username]/digital-business-brain
git push -u origin main

# Deploy
# 1. Visit vercel.com
# 2. Import GitHub repository
# 3. Deploy!
```

### Local Development
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
